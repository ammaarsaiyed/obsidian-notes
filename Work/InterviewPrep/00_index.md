# Lead Engineer Interview Prep - Split Index

This folder splits the original `../lead_engineer_interview_prep.md` into focused interview prep files.

## Recommended Use

1. Start with `01_role_positioning.md` for your core narrative.
2. Memorise the project examples in `02_project_stories.md`.
3. Use `05_technical_deep_dives.md` for Grade E technical depth.
4. Use `08_values_deep_dives.md` for detailed values/rubric answers.
5. Finish with `10_questions_to_ask_and_close.md` for questions and closing statements.

## Files

- `01_role_positioning.md` - role target, evidence map, opening answer, motivation and lead engineer readiness.
- `02_project_stories.md` - six reusable STAR project stories.
- `03_role_alignment_questions.md` - broad role-alignment prompts and answers.
- `04_technical_topic_answers.md` - concise technical topic answers.
- `05_technical_deep_dives.md` - detailed Grade E technical scripts.
- `06_system_design_practice.md` - system design prompts for agentic AI, RAG and monitoring.
- `07_values_answer_bank.md` - concise Lloyds values answer bank.
- `08_values_deep_dives.md` - detailed values scripts with steps and follow-ups.
- `09_quick_questions_and_followups.md` - quick answers and likely follow-up question bank.
- `10_questions_to_ask_and_close.md` - questions to ask them, first 90 days, close and checklist.
